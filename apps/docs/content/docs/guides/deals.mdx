title: Managing Deals
description: How to create, manage, and track deals through the pipeline.
---

Deals are the core unit of work in dscr.ai. Each deal represents a loan opportunity moving through your origination pipeline.

## Creating a Deal

Deals can be created through the UI or via the API. Each deal is associated with an organization and moves through configurable pipeline stages.

1. Navigate to the **Deals** page and click **New Deal**.
2. Select a loan program to pre-populate pricing rules (optional).
3. Fill in the property address, loan amount, and deal type.
4. Link borrowers and entities to the deal.

## Pipeline Stages

Configure custom pipeline stages to match your workflow. Deals progress through stages like Application, Underwriting, Approval, and Closing with automated task tracking at each step.

### Default Stages

| Stage | Description |
|-------|-------------|
| Application | Initial intake and data collection |
| Underwriting | Risk assessment and document review |
| Approval | Final review and decision |
| Closing | Document execution and funding |

## Pricing Scenarios

Each deal can have multiple pricing scenarios. Scenarios allow you to model different rate structures, terms, and fee configurations to present options to borrowers.

## Task Automation

Assign tasks to deal stages to ensure consistent workflows. Tasks can be auto-generated when a deal enters a stage, and team members receive notifications for assigned tasks.
