## New Features

- **Developer Documentation** — Rebuilt documentation with Fumadocs for improved developer experience, interactive API playground via Scalar, and built-in search.
- **API Contract Package** — Introduced `@repo/api-contract` with Zod schemas and OpenAPI generation for type-safe API development.
- **TypeScript & Python SDKs** — Official SDK packages with full type safety and IntelliSense support.

## Improvements

- Enhanced deal pipeline task tracking with customizable task templates.
- Improved document management workflows with bulk upload support.
- Added scenario comparison view for side-by-side pricing analysis.

## Bug Fixes

- Fixed pagination issues in the deals data table.
- Resolved authentication token refresh edge cases.
- Fixed document template variable substitution for nested entity data.
