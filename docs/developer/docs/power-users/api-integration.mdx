The dscr.ai API enables deep integrations with your existing systems — CRMs, loan origination systems, accounting platforms, and more.

## Integration Patterns

Common integration patterns include syncing deal data to your CRM, automating document generation from external systems, and pushing pricing updates from third-party rate feeds.

## API Clients

Use the official TypeScript or Python SDKs for type-safe API access, or call the REST API directly from any language.
