## March 2026

### New Features

- **Mintlify Documentation** — Migrated documentation to Mintlify Aspen for improved developer experience, interactive API playground, and better search.
- **API Contract Package** — Introduced `@repo/api-contract` with Zod schemas and OpenAPI generation for type-safe API development.

### Improvements

- Enhanced deal pipeline task tracking with customizable task templates.
- Improved document management workflows with bulk upload support.

### Bug Fixes

- Fixed pagination issues in the deals data table.
- Resolved authentication token refresh edge cases.
